# disklook-rs
A beautiful and minimalist CLI storage space visualizer written in Rust.

---

## Features

- Shows all partitions, disk kind, how many space is taken, how big is the disk and the usage in precent 

- Cross-platform (Android, FreeBSD, NetBSD, iOS, Linux, macOS, Raspberry Pi and Windows)

- Fast and lightweight

---

## Installation

You can easily install it with:
```bash
cargo install --path .
```
and execute with:
```bash
disklook
```
